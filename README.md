## 💫 Ce qui me motive

<table>
<tr>
<td width="50%" valign="top">

**🤖 Intelligence Artificielle**

L'IA est un levier concret, pas un buzzword. Je l'intègre dans chaque app pour automatiser les tâches répétitives, personnaliser l'expérience et résoudre des problèmes complexes avec élégance.

</td>
<td width="50%" valign="top">

**🚑 Impact Social**

Mes projets répondent à des besoins réels : sécurité, bien-être, productivité. Passionné de secourisme, je crois que la tech peut sauver du temps — et parfois bien plus.

</td>
</tr>
<tr>
<td width="50%" valign="top">

L'œil du photographe influence mon code : attention au détail, composition, esthétique. C'est ce qui rend mes interfaces soignées et mes UX vraiment intuitives.

</td>
<td width="50%" valign="top">

**🌍 Web Accessible**

Un web pour tous, pas pour quelques-uns. Je construis des interfaces claires, rapides, pensées pour l'humain — peu importe le device ou le niveau tech.
